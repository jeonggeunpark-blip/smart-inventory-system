# 📱 Field DX: Mobile Visual Reporting System

## 📖 プロジェクト概要 (Overview)
Google Apps Script (GAS) をバックエンドに採用し、**完全レスポンシブなSPA（シングルページアプリケーション）**として開発した現場管理システムです。
単なる数値管理だけでなく、「現場の状況」を視覚的に記録・共有することに特化しており、**カメラ撮影・写真への書き込み・QRスキャン**までをブラウザひとつで完結させました。

> **Concept:** "PCを開く時間をゼロにする"。現場作業者がスマホ片手にすべての報告業務を完了できるUI/UXを追求。

---

## 🛠 技術的構成 (Tech Stack & Architecture)
GASの制約（ロード速度など）を克服するため、現代的なWeb技術をフル活用しています。

* **Frontend:** HTML5, Tailwind CSS (Utility-first CSS)
    * *Dark Mode Support (OS設定に連動および手動切替)*
* **JavaScript:** Vanilla JS (SPA Router System)
    * *No Framework:* 軽量化のためReact等は使わず、独自のルーターで画面遷移を高速化
* **Libraries:**
    * `Chart.js`: リアルタイムな在庫・データの可視化
    * `jsQR`: ブラウザ上でのQRコード/バーコード解析
    * `Phosphor Icons`: 直感的なUIアイコン
* **Backend:** Google Apps Script (doGet / HTMLService)

---

## 💡 実装機能 (Key Features)

### 1. 📸 高度な画像処理 (Visual Documentation)
* **カメラ連動 & ギャラリーアップロード:** アプリ内から直接カメラを起動。
* **ドローイング機能 (Image Annotation):** 撮影した写真に、その場で「赤丸」や「矢印」を描き込み、問題箇所を直感的に報告可能（Canvas API活用）。

### 2. ⚡ 現場特化型入力 (Smart Input)
* **QRロケーション管理:** 棚や場所のQRコードをスキャンし、置き場情報を自動入力。
* **サジェスト機能:** 入力履歴から候補を自動表示し、タイピングの手間を極限まで削減。

### 3. 📊 ダッシュボード & KPI (Real-time Analytics)
* **警告アラート:** 在庫切れや異常値を自動検知し、赤字で強調表示。
* **多言語対応:** 日本語/英語 (i18n) の切り替えに対応し、外国人労働者がいる現場でも運用可能。

---

## 📈 DX導入効果 (Business Impact)
このシステムは、以下の「現場の無駄」を削減するために設計されました。

* **報告工数の削減:** 写真を撮って事務所に戻り、Excelに貼り付けて矢印を描く…という作業が**「その場で3秒」**に短縮。
* **教育コスト削減:** マニュアル不要の直感的なUI（スマホネイティブ風デザイン）により、高齢の作業者でも即日利用可能。
* **設備投資ゼロ:** 専用のハンディターミナルやタブレット購入が不要。個人のスマホや既存端末で即稼働。
