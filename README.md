# 🚀 Smart Inventory Manager (在庫管理・自動発注システム)

## 📖 プロジェクト概要 (Overview)
Google Apps Script (GAS) と AppSheet を活用して開発した、**スマートフォン完結型の在庫管理システム**です。
現場の「手書き管理・Excel入力」の手間を解消し、バーコードスキャン一つで入出庫管理から発注書作成までを自動化しました。

> **ポイント:** 高額なSaaSを導入せず、既存のGoogle Workspace環境のみで構築することで**導入コストゼロ**を実現しました。

---

## 💥 課題と背景 (Problem)
以前の現場では、以下のような課題がありました。
1.  **タイムラグ:** 在庫表への記入が後回しになり、実在庫とデータが合わない。
2.  **発注ミス:** 在庫切れに気づくのが遅れ、緊急発注や業務停止が発生していた。
3.  **属人化:** 特定の担当者しか在庫状況を把握していなかった。

## 💡 解決策 (Solution)
このシステムを導入することで、DX（デジタルトランスフォーメーション）を推進しました。

* **QR/バーコードスキャン:** スマホカメラで商品のバーコードを読み取るだけで入出庫を記録。
* **リアルタイム同期:** スプレッドシートへ即時反映され、誰でも最新在庫を確認可能。
* **自動アラート:** 在庫が設定した「発注点」を下回ると、Slack/LINEに自動通知。
* **ワンクリック発注:** 発注書PDFを自動生成し、メール下書きまでをワンストップで実行。

---

## 🛠 使用技術 (Tech Stack)
* **Frontend:** AppSheet (UI/UX), HTML/CSS (Web App Dashboard)
* **Backend:** Google Apps Script (GAS)
* **Database:** Google Spreadsheet
* **Communication:** Slack API / LINE Notify
